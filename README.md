# soundcloud-best

SoundCloud Best is a modified and enhanced SoundCloud desktop client built on Electron. The project expands the standard SoundCloud experience with features designed for active music listeners and Discord users.

Key additions include Discord Rich Presence integration, allowing users to display their currently playing tracks on Discord, as well as built-in lyrics support that can be shown directly within the application or through Rich Presence. The interface has also been redesigned to provide a cleaner and more customizable user experience compared to the standard SoundCloud client.
